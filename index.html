Skill

Belajar CSS

Saya belajar CSS dasar untuk membuat website agar tampil menarik. CSS digunakan untuk mengatur elemen seperti warna, font, dan tata letak, yang meningkatkan estetika dan keterbacaan situs. Dengan CSS, saya dapat dengan mudah memelihara dan mengubah desain tanpa mengubah HTML.

Pelatihan dengan IL

Saya belajar dan berlatih dengan Infinite Learning untuk mengembangkan keterampilan dalam membuat web fullstack. Program ini memberikan pemahaman mendalam tentang teknologi front-end dan back-end, memungkinkan saya untuk membangun aplikasi web yang fungsional dan menarik. Melalui pelatihan ini, saya siap menghadapi tantangan di dunia pengembangan web.

Pelatihan FO dengan TELKOM

Dalam pelatihan Fiber Optic (FO) dengan TELKOM, saya belajar cara instalasi jaringan serat optik yang efisien. Melalui sesi teori dan praktik, saya memahami langkah-langkah penting mulai dari pemilihan alat, persiapan kabel, hingga teknik penyambungan dan pengujian jaringan. Pelatihan ini tidak hanya memperdalam pengetahuan saya, tetapi juga memberikan keterampilan langsung yang dibutuhkan untuk menghadapi tantangan dalam dunia teknologi komunikasi saat ini. Pengalaman ini sangat berharga untuk karier saya di bidang jaringan.
